P3 Clock

ピー・スリー・クロック

.zip File
Link this javascript file to your page to add a clock element inspired by Persona 3.

MODIFIERS
img_directory - The directory where the required images are located.
element_parent - The element the clock is located in.
time_patterns - Specify the time label according to the hour.
day_name - Specify the names of the days of the week.

P4 Clock

ピー・フォー・クロック

.zip File
Link this javascript file to your page to add a clock element inspired by Persona 4.

MODIFIERS
img_directory - The directory where the required images are located.
element_parent - The element the clock is located in.
clock_sprite - The clock sprite to display.
weather_icon - Choose which weather icon will appear.
time_patterns - Specify the time label according to the hour.
day_name - Specify the names of the days of the week.
